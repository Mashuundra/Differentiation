Модуль дифференцирования математических выражений.

Содержит классы для вычисления производной и упрощения результата.

Дифференцирование — это процесс нахождения производной функции.
Модуль реализует автоматическое дифференцирование с помощью паттерна Visitor.

Основные компоненты:

1. DifferentiationVisitor — класс, реализующий правила дифференцирования
2. Simplifier — класс, упрощающий полученные выражения

Visitor (посетитель) — это поведенческий паттерн, который позволяет добавлять новые операции к структуре данных без изменения самих классов узлов.

Правила дифференцирования:
Константа:          d/dx (c) = 0
Переменная:         d/dx (x) = 1
Сумма:              (u + v)' = u' + v'
Разность:           (u - v)' = u' - v'
Произведение:       (u * v)' = u'*v + u*v'
Частное:            (u / v)' = (u'*v - u*v') / v²
Степень:            (x^n)' = n * x^(n-1)

Функции:
  sin(x)'  = cos(x)
  cos(x)'  = -sin(x)
  tan(x)'  = 1/cos²(x) = sec²(x)
  ln(x)'   = 1/x
  exp(x)'  = exp(x)

Для вложенной функции:
  f(g(x))' = f'(g(x)) * g'(x)


Упрощение выражений:
После дифференцирования результат часто требует упрощения.
Simplifier применяет алгебраические преобразования:

Арифметические упрощения:
  0 + a  → a
  a + 0  → a
  1 * a  → a
  a * 1  → a
  0 * a  → 0
  a * 0  → 0

Упрощение степеней:
  x^1  → x
  x^0  → 1
  x * x → x^2
  x * x^2 → x^3
  x^2 * x^3 → x^5

Упрощение знаков:
  -(-a) → a
  -0    → 0
  a + (-b) → a - b
  a - (-b) → a + b

Группировка множителей:
  x * 2 * y → 2 * x * y
  x * x * y → x^2 * y

Процесс упрощения выполняется многократно, пока выражение не перестанет меняться.
