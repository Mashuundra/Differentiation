AST модуль предоставляет гибкую и расширяемую архитектуру
для работы с математическими выражениями, позволяя легко добавлять новые операции
без изменения существующих классов узлов.

AST (Абстрактное Синтаксическое Дерево)
— это древовидное представление структуры математического выражения.
Каждый узел дерева представляет либо операцию, либо значение, либо переменную.

2*x + sin(x) :

        BinaryOpNode(+)
        /            \
    BinaryOp(*)    FunctionNode(sin)
    /        \           |
Number(2)  Variable(x) Variable(x)


Visitor — это поведенческий паттерн, который позволяет добавлять
новые операции к структуре данных без изменения самих классов узлов.

у нас есть дерево AST и если нужно:
Вычислить производную, упростить выражение, преобразовать в строку и тп..
Без Visitor пришлось бы добавлять методы derivative(), simplify(), to_string()
в каждый класс узла.
